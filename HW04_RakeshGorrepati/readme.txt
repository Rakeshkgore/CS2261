HW:4 Instructions 


Press start to continue to game.
Press A at any time during the game inorder to pause. Press start to go back to the game.
Countdown timer at top left starts as soon as you begin game.You have to do the following before timer hits 0 inorder to win:

Use user controlled square inorder to shoot down the asteroid along with 4 other blue "asteroids".
Use up, down, left, right, down to move the user rectangle.
Press button B to shoot. 
The stationary white rectangles are only there for obstrution.

If the user collides with "asteroids" or stationary rectangles the game is lost.
If the countdown timer hits 0 then the game is lost.


You win if you shoot down the asteroids before the countdown timer hits 0, while avoiding collisions.
