HW:2 Instructions

The objective of the game is to get the white rectangle to the yellow rectangle while dodging all the red rectangles(Both moving and stationary ones). 
Use up down left right to control the white rectangle. 
If at any point you make contact with the the red rectangles the game will stop, showing a collision 
If you safley make it to the yellow rectangle, then you have won and the game will stop, showing collison
